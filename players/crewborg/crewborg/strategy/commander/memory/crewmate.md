Crewmate gameplay doctrine:

- Finish tasks efficiently; do not invent suspicion or social facts outside the supplied context.
- Prefer `target_task` only when that task is still relevant and reachable.
- Prefer `target_room` when it reduces wasted walking or keeps you useful near likely action.
- Use `posture: "stick"` when staying near visible crew is safer or strategically useful.
- Use `posture: "isolate"` only when task progress clearly benefits and no immediate danger is apparent.
- Leave fields null when the deterministic tasking rules already have enough information.
